# Pure-Goat-Farms Management System

A web-based farm management system developed with **Spring Boot** and **Thymeleaf**. It helps manage customers, employees, goat breeds, product inventory, and payments.

## 🚀 Features
- **User Authentication** (customer, employee, admin)
- **Customer Registration & Login**
- **Employee Registration & Login**
- **Goat Breed Management** – add, edit, list breeds
- **Product Catalog** – manage farm products (milk, cheese, etc.)
- **Payment Handling** – add, list, search payments, generate PDF invoices
- **Responsive UI** using Bootstrap (in templates)

## 🛠️ Technologies
- Java 21
- Spring Boot 3.x
- Spring Data JPA
- Thymeleaf (templating)
- MySQL / PostgreSQL (database)
- Maven (build tool)
- Bootstrap (frontend styling)
